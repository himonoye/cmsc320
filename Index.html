Hypothesis Testing:¶

What is Hypothesis Testing? a hypothesis that is testable on the basis of observing a process that is modeled via a set of randomvariables.

https://en.wikipedia.org/wiki/Statistical_hypothesis_testing

Steps to Hypothesis Testing:

  • Find a Null Hypothesis. It's usually the opposite of your alternative hypothesis
  • state your Null Hypothesis
  • Find a test to test your Hypothesis
  • Use the result to either reject Null Hypothesis(accept alternative hypothesis) or accept Null Hypothesis

Gradient Descent¶

Below is a basic implementation of a gradient descent algorithm. The basic idea is to find the negative of the gradient(slop) of the beginning spot, and then add the value in order to move on to the next point and calculate the negative of the gradient again to reach the next spot. The function could be used to find the local maximum.

# Training data (X, y), T time steps, alpha step
def grad_descent_logreg(X, y, T, alpha):
    m, n = X.shape # m = #examples, n = #features
    theta = np.zeros(n)    # initialize parameters
    f = np.zeros(T)        # track loss over time
    for i in range(T):
        h = np.exp(X.dot(theta))/(1+np.exp(X.dot(theta)))
        # loss for current parameter vector theta
        f[i] = np.log2(1+np.exp(h.dot(y)))
        # compute steepest ascent at f(theta)
        g = (y-h).dot(X) 
        # step down the gradient
        theta = theta - alpha*g
    return theta, f
